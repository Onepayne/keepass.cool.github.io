Pourquoi KeePass ?

Aujourd'hui, vous devez mémoriser de nombreux mots de passe. Vous avez besoin d'un mot de passe pour de nombreux sites Web, votre compte de messagerie, votre serveur Web, vos identifiants réseau, etc. La liste est infinie. De plus, vous devez utiliser un mot de passe différent pour chaque compte, car si vous n'utilisez qu'un seul même mot de passe partout et que quelqu'un obtient ce mot de passe, alors vous aurez un problème : le voleur aura accès à tous vos comptes.

KeePass est un gestionnaire de mots de passe open source gratuit, qui vous aide à gérer vos mots de passe de manière sécurisée. Vous pouvez stocker tous vos mots de passe dans une base de données, qui est verrouillée avec une clé principale. Vous n'avez donc besoin de mémoriser qu'une seule clé principale pour déverrouiller toute la base de données. Les fichiers de base de données sont chiffrés à l'aide des meilleurs algorithmes de cryptographie les plus sûrs actuellement connus (AES-256, ChaCha20 et Twofish). Pour plus d'informations, consultez la page des fonctionnalités.

Est-ce vraiment gratuit ?

Oui, KeePass est vraiment gratuit, et plus que cela : il est open source (certifié OSI) ! Vous pouvez consulter son code source dans son intégralité et vérifier si les fonctionnalités de sécurité sont correctement implémentées.

Pour KeePass 1.x : placer le fichier d'aide locale en décompressant l'archive ci-dessus dans le répertoire où se situe le fichier KeePass.exe. Puis menu Aide -> Origine de l'aide... -> Fichier d'aide locale. Il ne suffit plus que d'appuyer sur la touche F1 du clavier une fois le logiciel KeePass lancé pour avoir l'aide en local.

Pour KeePass 2.x : placer le fichier d'aide locale en décompressant l'archive ci-dessus dans le répertoire où se situe le fichier KeePass.exe. Puis sélectionner le choix 'Fichier d'aide locale' dans le logiciel KeePass comme indiqué ci-après sur la capture d'écran. Il ne suffit plus que d'appuyer sur la touche F1 du clavier une fois le logiciel KeePass lancé pour avoir l'aide en local.

N.B. : penser à débloquer, à l'aide de l'explorateur de fichiers, le fichier KeePass.chm dans ses propriétés, s'il n'est pas correctement lisible à l'écran.
